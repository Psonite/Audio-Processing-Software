Stock presets

Every plug-in comes with a selection of stock presets which can be opened within your softwareâ€™s built-in preset viewer. These presets are hard coded into each plug-in so no extra installation is required.

Opening presets

Firstly, I recommend you create a folder for your presets so you can keep everything organised. In Ableton, you navigate to the plugin and click on the folder icon. This will bring up a window where you can select the preset you want to load.

Saving presets

When saving presets in Ableton, navigate to the save icon and click on it. This will bring up a window where you can rename and save your preset.

Chroma Audio is an ever-growing collection of plug-ins aimed at providing high-quality and easy to use software for processing audio. The Essentials collection is the most recent addition to the project consisting of 4 plugins; Delay, Distortion, Flanger and a Compressor. Each plugin in this collection has been based off guitar pedals leading to a professional, easy to use interface.
